3. Goal 生命周期

🚀 创建

/goal 学习Python
→ append_goal()
→ 写入 goals.json
→ ensure_goal_dir()
创建 memory/ experiences/ sessions/ 骨架
→ 限制：最多 5 个活跃目标
→

🎯 进入目标模式

/goal switch GOAL-xxx
或 Tab 空提示符循环
→ GoalAgentContext.switch_to()
→ 保存旧会话快照
→ 关联旧会话到 goal
→ 清空对话 → 新 session
→ Layer 4 注入到 System Prompt
提示符：ohmy [学习Python] >
所有 /exper 默认写入 goal 专属库
→

🚪 退出 / 切换

/goal exit 或 Tab 循环
→ _has_meaningful_conversation()
检查是否有实际问答
→ 有内容：弹出确认
「当前对话有新的内容，是否更新目标记忆？ [Y/n]」
→ Y → /goal memory update
→ 清空对话 → 新 session

4. 记忆更新流程 (AI 专属)

仅 AI 可写入长期记忆。人类通过 /exper add 写入经验库。
触发方式：/goal memory update 命令 或 退出目标时自动询问。

⚡ 触发

① /goal memory update
手动触发
② /goal exit
退出时自动询问
③ Tab 循环切换
离开当前 goal 时询问
④ AI 自主调用工具
file_edit / file_write 写入 memory/
→

🔄 处理

update_goal_memory_via_ai()
1. 读取已有记忆文件
2. 提取对话文本
3. 组装 prompt（含 goal 信息 + 已有记忆 + 对话）
4. 调 LLM 分析 → 生成三部分更新
5. 解析响应 → 追加到对应文件
→

📝 输出 (3 文件)

ai_notes.md
新进度 / 关键决策 / 教训
user_prefs.md
学习到的用户偏好 / 约束
context.md
重要事实 / 项目状态
每部分内容为追加模式（不覆盖已有内容）。
如果某部分无新内容，LLM 返回 "NONE"。

5. 记忆检索流程 (Subagent)

Agent 在决策时可调用 delegate_task("goal_memory_retriever") 检索记忆。
用户也可通过 /goal memory search [query] [--deep] 手动触发。

🔍 默认范围（快速）

检索 memory/ai_notes.md
检索 memory/user_prefs.md
检索 memory/context.md
检索 experiences/default.md
✓ 仅检索提炼后的内容
✓ 速度快，精准度高

🔬 Deep 模式 --deep

（在默认基础上额外搜索）
检索 sessions/summaries/
检索 data/sessions/ 原始快照
Fallback 全局经验库
⚠ 数据量大，噪音多
⚠ 仅在需要回溯原始对话时使用

📋 检索报告格式

Subagent 返回结构化报告：
- 来源文件：完整路径
- 记忆/经验 ID：EXP-xxx 或条目标题
- 原文摘录：相关内容片段
- 相关原因：为什么与查询相关
- 相关度：高 / 中 / 低

6. 每日计划流程

📥 捕捉

/plan [内容]
→ append_plan()
→ 写入 plans/YYYY-MM-DD.inbox.md
→

🤖 AI 整理

触发 _stream_prompt()
→ AI 读取 inbox + 活跃 Goal 列表
→ 按 Goal topic 分组
→ 检查未覆盖目标的长期事项
→ 写入 plans/YYYY-MM-DD.md
→

📋 展示

/plan（无参数）
→ 读取当天整理后的计划
→ 按 Goal 分组渲染为 Markdown
inbox 保留原始条目，不删除

8. System Prompt 四层结构

通用模式

# Layer 1: Base Prompt
基础人格：长期成长助手，48行中文 prompt
# Layer 2: User Prompt
用户自定义系统提示词
# Layer 3: Model / User Profile
model_profile.md + user_profile.md
# Environment
OS / Shell / CWD / Date / Git ...

Goal 模式（+ Layer 4）

Layer 1 + 2 + 3（同上）
# Environment
# 当前专注目标
- 目标: {topic}
- 描述: {description}
- 进度: {progress}% | 截止: {ends_at}
# 目标 AI 笔记
最近 30 行 ai_notes.md
# 目标用户偏好
最近 20 行 user_prefs.md
# 最近相关会话
最近 3 个关联 session 摘要
# 目标模式说明
经验查询默认优先搜索本目标专属库
会话自动关联到此目标
/goal exit 退出，Tab 键循环切换
